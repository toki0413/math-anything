### 2.3 Morse理论 + 拓扑分析

```
┌─────────────────────────────────────────────────────────────────┐
│              Morse-同调理论 (Morse-Homology)                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  能量景观: V: M → ℝ                                             │
│                                                                 │
│  Morse理论:                                                     │
│  - 临界点: ∇V = 0                                               │
│  - Hessian: H = ∂²V/∂qᵢ∂qⱼ                                      │
│  - Morse指数: μ = #负特征值                                     │
│                                                                 │
│  Morse不等式:                                                   │
│  #临界点的k阶部分 ≥ βₖ(M)                                       │
│                                                                 │
│  Morse同调:                                                     │
│  Cₖ = 由μ=k的临界点生成的自由群                                 │
│  ∂: Cₖ → Cₖ₋₁                                                   │
│  HMₖ = ker ∂ / im ∂ ≅ Hₖ(M)                                     │
│                                                                 │
│  应用: 能量景观的拓扑分析                                        │
│  - 稳定态: μ = 0 的临界点                                       │
│  - 过渡态: μ = 1 的临界点 (鞍点)                                │
│  - 反应路径: 梯度流线                                           │
│                                                                 │
│  示例: 双阱势能                                                  │
│  V(x) = (x²-1)²                                                 │
│  临界点: x = -1 (μ=0), x = 0 (μ=1), x = 1 (μ=0)                 │
│  同调: H₀ = ℤ², H₁ = 0                                          │
│  Morse不等式:                                                   │
│  #μ=0 = 2 ≥ β₀ = 2 ✓                                            │
│  #μ=1 = 1 ≥ β₁ = 0 ✓                                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 统一的工作流

```python
class UnifiedMathFramework:
    """统一数学框架"""
    
    def __init__(self):
        self.symplectic_integrator = None
        self.constraint_manifold = None
        self.topology_analyzer = None
        self.morse_analyzer = None
        self.latent_space = None
    
    def analyze(self, input_file: str):
        """完整的五层分析"""
        
        # Layer 1: 辛结构识别
        symplectic_info = self._identify_symplectic_structure(input_file)
        
        # Layer 2: 约束流形识别
        manifold_info = self._identify_constraint_manifold(input_file)
        
        # Layer 3: 拓扑分析
        topology_info = self._analyze_topology(manifold_info)
        
        # Layer 4: Morse理论分析
        morse_info = self._analyze_morse_theory(manifold_info)
        
        # Layer 5: 潜空间设计
        latent_info = self._design_latent_space(
            manifold_info, topology_info, morse_info
        )
        
        return UnifiedAnalysisResult(
            symplectic=symplectic_info,
            manifold=manifold_info,
            topology=topology_info,
            morse=morse_info,
            latent=latent_info
        )
    
    def _design_latent_space(self, manifold, topology, morse):
        """基于流形、拓扑、Morse设计潜空间"""
        
        # 维度确定
        # d = β₀ + β₁ + 2*(#过渡态)  # 启发式公式
        d = topology.betti_0 + topology.betti_1 + 2 * morse.num_saddles
        d = max(16, min(d, 128))  # 限制范围
        
        # 结构确定
        structure = {
            "continuous": topology.betti_0,  # 每个连通分支的坐标
            "cyclic": topology.betti_1,       # 每个环的相位
            "barrier": morse.num_saddles      # 每个能垒的参数
        }
        
        return LatentSpaceDesign(
            dimension=d,
            structure=structure,
            equivariance=self._determine_equivariance(),
            symplectic=self._check_symplectic_requirement(),
            manifold_constraint=True
        )
```
