# Change Log

All notable changes to the "Bourbaki" extension will be documented in this file.

## [0.1.0]

- Initial release
- Syntax highlighting for VASP INCAR/POSCAR, Quantum ESPRESSO, LAMMPS, Abaqus, OpenFOAM, GROMACS
- Auto-completion and hover documentation for common engine parameters
- Bourbaki mathematical structure analysis via LSP
- Workspace engine detection
- Equation verification and numerical solver invocation
